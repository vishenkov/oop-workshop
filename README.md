# get-geo cli utility

## Usage
### Run from console
```
get-geo 127.0.0.1
```

### Run locally
```
make run
```

### Build
```
make build
```

### Setup
```
yarn
```
OR
```
make install
```

### Tests
```
make test
```

### pipeline
```
make run-pipe
```

Based on [hexlet](https://hexlet.io) boilerplate [nodejs-package](https://github.com/hexlet-boilerplates/nodejs-package)
